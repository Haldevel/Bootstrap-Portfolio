Furry Friends Finder

This full-stack application finds three top mathing small breeds of dogs based on a user's answers to five survey questions. The app displays the breeds and the images of these best matching small breeds. The app uses Express to handle routing.

what2grow

This is a web application designed to help amateur gardeners make informed decisions about best time to grow plans. The app provides the first suitable month and the optimal month for the plant to grow based on the user's geolocation, as well on the historical daily min, max, and mean temperatures for 2015-2018 years provided by aWhere API.

Myazaki Moments

The web app uses Giphy API to retrieve images of Miyazaki movies, which names are displayed on the buttons generated by the app. When a user clicks on each gif, the source of the image is switched to the animated version of the gif, and that gives the impression that the image moves on a mouse click.

With help of autocomplete, the user types in a crop or plant of their choice.
